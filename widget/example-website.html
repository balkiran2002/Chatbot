How to Use the Chatbot

  1. Locate the chat icon - Look for the floating chat bubble in the bottom right corner
  2. Click to open - Click the icon to open the chat window
  3. Ask a question - Type your question and press Enter or click Send
  4. Get an answer - The chatbot will match your question with our FAQ database and provide an answer
  5. Contact support - If your question isn't answered, we'll provide you with support contact information

Integration Code

This is the code used to integrate the chatbot into this website:

<!-- Load the chatbot widget -->
<script src="http://localhost:5000/chatbot-widget.js"></script>

<!-- Initialize the chatbot -->
<script>
  ChatbotWidget.init({
    apiUrl: 'http://localhost:5000',
    position: 'bottom-right', // or 'bottom-left'
    theme: 'light' // or 'dark'
  });
</script>
📌 Demo Note: This page is running locally. In production, make sure to:
  • Deploy the backend API to a server
  • Update the API URL in the widget initialization
  • Enable CORS for your domain
  • Serve the widget over HTTPS
